# Gossip Girl App 💋

This is a digital Gossip Girl-style app made with [Rork](https://rork.com) — no-code platform for creating apps.

## 💡 Features
- User profiles for each character
- Gossip feed system (like a blog or anonymous submission)
- Aesthetic UI based on NYC vibes
- Chat & message simulation
- Fake rumors functionality
